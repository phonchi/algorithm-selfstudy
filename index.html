MERGE-SORT(A, p, r) if p >= r return q = floor((p + r) / 2) MERGE-SORT(A, p, q) MERGE-SORT(A, q + 1, r) MERGE(A, p, q, r) RELAX(u, v, w) if v.d > u.d + w(u, v) v.d = u.d + w(u, v) v.pi = u
